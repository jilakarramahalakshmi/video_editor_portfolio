# 🎬 AS Editor Portfolio

A professional Video Editor Portfolio built with React, Vite, Firebase Authentication, Firestore, and Firebase Storage. This website showcases video editing projects, posters, and categories while providing a secure admin dashboard for managing content.

## ✨ Features

* 🎥 Video editing portfolio
* 🖼️ Poster gallery
* 📂 Category-wise projects
* 🔐 Firebase Authentication (Admin Login)
* 📤 Secure project upload
* 📱 Responsive design
* ⚡ Built with React + Vite
* ☁️ Firebase Firestore & Storage integration

## 🛠️ Tech Stack

* React
* Vite
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* React Router DOM
* CSS

```bash

npm install
npm run dev
```

## 📦 Build

```bash

```
